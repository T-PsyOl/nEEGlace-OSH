# nEEGlace GUI
The GUI is developed and maintained separately [here](https://github.com/T-PsyOl/nEEGlace-GUI). Make sure to always use the latest version from this repository.
The GUI application for nEEGlace device is built with CustomTkinter, PyQt5 and LSL.  

The GUI helps to:
1. Configure the EEG Device
2. Configure the audio detection parameters for BELA
3. Real-time visualisation of the EEG signal, Sound trigger and real-time ERP plotting 

## Installation Guide

```bash
git clone https://github.com/T-PsyOl/nEEGlace-GUI.git
cd nEEGlace-GUI
pip install -r requirements.txt
```

## Run GUI

```bash
python run.py
```
