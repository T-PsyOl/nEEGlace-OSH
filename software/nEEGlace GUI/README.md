# nEEGlace GUI
[IMPORTANT] The GUI is developed and maintained separately [here](https://github.com/T-PsyOl/nEEGlace-GUI/blob/main/README.md). Make sure to always use the latest version from this repository.
The GUI application for nEEGlace device is built with CustomTkinter, PyQt5 and LSL for EEG device communication and real-time visualisation.

## Installation Guide

```bash
git clone https://github.com/T-PsyOl/nEEGlace-GUI.git
cd nEEGlace-GUI
pip install -r requirements.txt
```

## Run GUI

```bash
python run.py
```
